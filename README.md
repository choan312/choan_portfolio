## üìù Prerequisites

- Node.js and npm: https://nodejs.org/
- Angular CLI: https://cli.angular.io/

## üõ† Installation

1. Clone the repository

   ```sh
   git clone https://github.com/choan312/choan_portfolio.git
   cd choan_portfolio
   ```

2. Install the dependencies

   ```sh
   npm install
   ```

3. Start the development server

   ```sh
   ng serve
   ```
